# GSAP Learning

## Project Config

### Install React

```bash
npm create vite@latest
```

### Install React Router

```bash
npm i react-router
```

### Install GSAP

```bash
npm install gsap @gsap/react
```

### Install Tailwind CSS via

```bash
npm install tailwindcss @tailwindcss/vite
```

### Run Project

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
